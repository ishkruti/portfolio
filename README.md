A clean, responsive personal portfolio built entirely with core web technologies.

Project URL : 
[https://ishkruti.github.io/portfolio/](https://ishkruti.github.io/portfolio/)

Project Challenge : 
This project was built as part of the [Single Page CV](https://roadmap.sh/projects/single-page-cv) challenge from roadmap.sh.

Tech Stack : 
* HTML5: Structured using semantic elements for better accessibility.
* CSS3: Custom styling including Flexbox/Grid for layout and Media Queries for responsiveness.

Key Features : 
* No Frameworks: Built with pure HTML and CSS for maximum performance and compatibility.
* Live Deployment: Hosted via GitHub Pages.
* Responsive Design: Fully optimized for mobile, tablet, and desktop screens.
